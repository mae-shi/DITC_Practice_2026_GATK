## 2.　BWAでマッピング & samtoolsでソート

02_bwa.sh
```
#!/bin/bash
set -euo pipefail

# === 引数 ===
SAMPLE="$1"
RUNID="$2"
R1="$3"
R2="$4"
REF="$5"

# === 設定 ===
THREADS=8
OUT="${SAMPLE}.sorted.bam"
LOG="bwa_${SAMPLE}.log"

# === 実行===
bwa mem -t "${THREADS}" \
    -R "@RG\tID:${RUNID}\tSM:${SAMPLE}\tPL:ILLUMINA" \
    "${REF}" \
    "${R1}" "${R2}" 2>> "${LOG}" | \
    samtools sort -@ "${THREADS}" -o "${OUT}" 2>> "${LOG}"

# インデックス作成
samtools index "${OUT}" 2>> "${LOG}"
```

実行 (.sorted.bam ,.sorted.bam.baiできる)
```
./02_bwa.sh \
NA18939 \
SRR1295537 \
SRR1295537_1.fastq.gz \
SRR1295537_2.fastq.gz \
~/Desktop/DICP/20260403_GATK_brew/reference/hg38/Homo_sapiens_assembly38.fasta
```

## 3.　MarkDuplicates で 重複リードを検出

03_markduplicates.sh
```
#!/bin/bash
set -euo pipefail

# === 設定 ===
THREADS=8
SAMPLE=$1

IN_BAM="${SAMPLE}.sorted.bam"
OUT_BAM="${SAMPLE}.dedup.bam"
METRICS="${SAMPLE}.dup_metrics.txt"
LOG="markdup_${SAMPLE}.log"

# === 実行 ===
gatk MarkDuplicates \
-I "$IN_BAM" \
-O "$OUT_BAM" \
-M "$METRICS" \
2>> "$LOG"

# インデックス作成
samtools index "$OUT_BAM" 2>> "$LOG"
```

## 4.　BaseRecalibrator で クオリティスコアの偏りを検出し、修正量を計算。

```
dbSNP
```
wget ftp://gsapubftp-anonymous@ftp.broadinstitute.org/bundle/hg38/dbsnp_146.hg38.vcf.gz
wget ftp://gsapubftp-anonymous@ftp.broadinstitute.org/bundle/hg38/dbsnp_146.hg38.vcf.gz.tbi
```

04_BR.sh
```
#!/bin/bash
set -euo pipefail

# === 設定 ===
SAMPLE=$1

REF=~/Desktop/DICP/20260403_GATK_brew/reference/hg38/Homo_sapiens_assembly38.fasta
DBSNP=~/Desktop/DICP/20260403_GATK_brew/reference/hg38/dbsnp_146.hg38.vcf.gz
MILLS=~/Desktop/DICP/20260403_GATK_brew/reference/hg38/Mills_and_1000G_gold_standard.indels.hg38.vcf.gz

IN="${SAMPLE}.dedup.bam"
OUT="${SAMPLE}.recal.table"
LOG="baserecal_${SAMPLE}.log"

# === 実行 ===
gatk BaseRecalibrator \
    -R "$REF" \
    -I "$IN" \
    --known-sites "$DBSNP" \
    --known-sites "$MILLS" \
    -O "$OUT" \
    2>> "$LOG"
```

実行 (.recal.table できる)
```
./04_BR.sh NA18939
```

## 5.　ApplyBQSR で BaseRecalibratorで算出した補正テーブルで塩基品質スコアを再調整

05_applybqsr.sh
```
#!/bin/bash
set -euo pipefail

# === 設定 ===
SAMPLE=$1

REF=~/Desktop/DICP/20260403_GATK_brew/reference/hg38/Homo_sapiens_assembly38.fasta
IN_BAM="${SAMPLE}.dedup.bam"
RECAL_TABLE="${SAMPLE}.recal.table"
OUT_BAM="${SAMPLE}.recal.bam"
LOG="applybqsr_${SAMPLE}.log"

# ===== 実行 =====
gatk ApplyBQSR \
    -R "$REF" \
    -I "$IN_BAM" \
    --bqsr-recal-file "$RECAL_TABLE" \
    -O "$OUT_BAM" \
    2>> "$LOG"

# インデックス作成
samtools index "$OUT_BAM"
```

実行 (.recal.bam, .recal.bam.bai できる)
```
./05_applybqsr.sh NA18939
```

## 6.　HaplotypeCaller で 変異　(variant)検出
VCF（変異リスト）出す

06_haplotypecaller.sh
```
#!/bin/bash
set -euo pipefail

# === 設定 ===
SAMPLE=$1 
THREADS=8

REF=~/Desktop/DICP/20260403_GATK_brew/reference/hg38/Homo_sapiens_assembly38.fasta
IN_BAM="${SAMPLE}.recal.bam"
OUT="${SAMPLE}.g.vcf.gz"
LOG="haplotypecaller_${SAMPLE}.log"

# === 実行 ===
gatk --java-options "-Xmx16G" HaplotypeCaller \
-R "$REF" \
-I "$IN_BAM" \
-O "$OUT_GVCF" \
-ERC GVCF \
--native-pair-hmm-threads "$THREADS" \
2>> "$LOG"

# インデックス作成
gatk IndexFeatureFile -I "$OUT_GVCF" 2>> "$LOG"
```

実行 (.g.vcf.gz, .g.vcf.gz.tbi できる)
```
./06_haplotypecaller.sh NA18939
```

## 7.　GenotypeGVCFs で　ジェノタイピング （gVCF → VCF）
gVCF形式の変異候補を最終的な変異として確定

07_genotypegvcfs.sh
```
#!/bin/bash
set -euo pipefail

# === 設定 ===
SAMPLE=$1
REF=~/Desktop/DICP/20260403_GATK_brew/reference/hg38/Homo_sapiens_assembly38.fasta

IN_GVCF="${SAMPLE}.g.vcf.gz"
OUT_VCF="${SAMPLE}.vcf.gz"
LOG="genotypegvcfs_${SAMPLE}.log"

# === 実行 ===
gatk --java-options "-Xmx16G" GenotypeGVCFs \
    -R "$REF" \
    -V "$IN_GVCF" \
    -O "$OUT_VCF" \
    2>> "$LOG"

# インデックス作成
gatk IndexFeatureFile -I "$OUT_VCF" 2>> "$LOG"
```

実行 (.vcf.gz, .vcf.gz.tbi できる)
```
./07_genotypegvcfs.sh
```

単一サンプル解析のため、VQSRではなくハードフィルタリング（Macのシリコンのため、CNNは使えない）

SNPとINDELの分離
```
gatk SelectVariants \
  -R Homo_sapiens_assembly38.fasta \
  -V NA18939.vcf.gz \
  --select-type-to-include SNP \
  -O NA18939.snps.vcf.gz

gatk SelectVariants \
  -R Homo_sapiens_assembly38.fasta \
  -V NA18939.vcf.gz \
  --select-type-to-include INDEL \
  -O NA18939.indels.vcf.gz
```

SNPフィルタリング
```
gatk VariantFiltration \
  -R Homo_sapiens_assembly38.fasta \
  -V NA18939.snps.vcf.gz \
  -O NA18939.filtered.snps.vcf.gz \
  --filter-name "QD_lt2" --filter-expression "QD < 2.0" \
  --filter-name "FS_gt60" --filter-expression "FS > 60.0" \
  --filter-name "MQ_lt40" --filter-expression "MQ < 40.0" \
  --filter-name "SOR_gt3" --filter-expression "SOR > 3.0" \
  --filter-name "MQRankSum_lt-12.5" --filter-expression "MQRankSum < -12.5" \
  --filter-name "ReadPosRankSum_lt-8" --filter-expression "ReadPosRankSum < -8.0"
