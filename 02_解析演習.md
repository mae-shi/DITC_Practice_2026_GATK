## 2.　BWAでマッピング & samtoolsでソート

```
bwa mem -t 4 \
    -R "@RG\tID:SRR1295537\tSM:NA18939\tPL:ILLUMINA" \
    Homo_sapiens_assembly38.fasta \
    SRR1295537_1.fastq.gz SRR1295537_1.fastq.gz 2>>　bwa_NA18939.log | \
    samtools sort -@ 4 -o NA18939.sorted.bam 2>>　bwa_NA18939.log | \
    samtools index NA18939.sorted.bam 2>> bwa_NA18939.log
```
・&emsp;-R で、Read Groupを付与  
・&emsp;2>>　.log  でログ出す  
・&emsp;samtools sort　で、bamファイルをソート  
・&emsp;samtools index で、ソートしたbamファイルのインデックス作成  

Read Group の意味  
&emsp; @RG                ← Read Group開始  
&emsp; ID:SRR1295537       ← run識別子  
&emsp; SM:NA18939         ← サンプル名  
&emsp; PL:ILLUMINA        ← シーケンサー種

## 4.　BaseRecalibrator でknown-sitesをもとに、塩基スコアをだす

まず、known-sitesをダウンロード

Mills (indel用)
```
wget ftp://gsapubftp-anonymous@ftp.broadinstitute.org/bundle/hg38/Mills_and_1000G_gold_standard.indels.hg38.vcf.gz
wget ftp://gsapubftp-anonymous@ftp.broadinstitute.org/bundle/hg38/Mills_and_1000G_gold_standard.indels.hg38.vcf.gz.tbi
```
dbSNP (SNP用)
```
wget ftp://gsapubftp-anonymous@ftp.broadinstitute.org/bundle/hg38/dbsnp_146.hg38.vcf.gz
wget ftp://gsapubftp-anonymous@ftp.broadinstitute.org/bundle/hg38/dbsnp_146.hg38.vcf.gz.tbi
```

BaseRecalibrator実行  
(.recal.table できる)  
```
gatk BaseRecalibrator \
    -R Homo_sapiens_assembly38.fasta \
    -I NA18939.dedup.bam \
    --known-sites dbsnp_146.hg38.vcf.gz \
    --known-sites Mills_and_1000G_gold_standard.indels.hg38.vcf.gz \
    -O NA18939.recal.table \
    2>> baserecal_NA18939.log
```

<br>

## 5.　ApplyBQSR で BaseRecalibratorで算出したテーブルで塩基品質スコアを再調整

```
gatk ApplyBQSR \
    -R Homo_sapiens_assembly38.fasta \
    -I NA18939.dedup.bam \
    --bqsr-recal-file NA18939.recal.table \
    -O NA18939.recal.bam 2>> applybqsr_NA18939.log

# インデックス作成

```

## 6.　HaplotypeCaller で 変異　(variant)検出
VCF（変異リスト）出す。(.g.vcf.gz, .g.vcf.gz.tbi できる)

```
gatk --java-options "-Xmx16G" HaplotypeCaller \
    -R Homo_sapiens_assembly38.fasta \
    -I NA18939.recal.bam \
    -O NA18939.g.vcf.gz \
    -ERC GVCF \
    --native-pair-hmm-threads 4 2>> haplotypecaller_NA18939.log

# インデックス作成
gatk IndexFeatureFile -I NA18939.g.vcf.gz 2>> haplotypecaller_NA18939.log
```

## 8.フィルタリング
VQSRではなくハードフィルタリング（Macのシリコンのため、CNNは使えない）

ハードフィルタリングのパラメーターは下記を参照  
[Filtering recommendations](https://gatk.broadinstitute.org/hc/en-us/articles/360037499012-I-am-unable-to-use-VQSR-recalibration-to-filter-variants#:~:text=the%20filtering%20parameters.-,Filtering%20recommendations,-Here%20are%20some)

SNPとINDELの分離
```
# SNP抽出
gatk SelectVariants \
  -R ~/Practice/GATK_20260403/reference/hg38/Homo_sapiens_assembly38.fasta \
  -V NA18939.vcf.gz \
  --select-type-to-include SNP \
  -O NA18939.snp.vcf.gz

# INDEL抽出
gatk SelectVariants \
  -R ~/Practice/GATK_20260403/reference/hg38/Homo_sapiens_assembly38.fasta \
  -V NA18939.vcf.gz \
  --select-type-to-include INDEL \
  -O NA18939.indel.vcf.gz

# SNPフィルタリングの実行
gatk VariantFiltration \
  -R ~/Practice/GATK_20260403/reference/hg38/Homo_sapiens_assembly38.fasta \
  -V NA18939.snp.vcf.gz \
  -O NA18939.snp.hf.vcf.gz  \
  --filter-name "QD_lt2" --filter-expression "QD < 2.0" \
  --filter-name "MQ_lt40" --filter-expression "MQ < 40.0" \
  --filter-name "FS_gt60" --filter-expression "FS > 60.0" \
  --filter-name "SOR_gt3" --filter-expression "SOR > 3.0" \
  --filter-name "MQRankSum_lt-12.5" --filter-expression "MQRankSum < -12.5" \
  --filter-name "ReadPosRankSum_lt-8" --filter-expression "ReadPosRankSum < -8.0"

# INDELフィルタリングの実行
gatk VariantFiltration \
  -R ~/Practice/GATK_20260403/reference/hg38/Homo_sapiens_assembly38.fasta \
  -V NA18939.indel.vcf.gz \
  -O NA18939.indel.hf.vcf.gz \
  --filter-name "QD_lt2" --filter-expression "QD < 2.0" \
  --filter-name "ReadPosRankSum_lt-20" --filter-expression "ReadPosRankSum < -20.0" \
  --filter-name "InbreedingCoeff_lt-08" --filter-expression "InbreedingCoeff < -0.8" \
  --filter-name "FS_gt200" --filter-expression "FS > 200.0" \
  --filter-name "SOR_gt10" --filter-expression "SOR > 10.0"

# SNPとINDEL合わせる
gatk MergeVcfs \
-I NA18939.snp.hf.vcf.gz \
-I NA18939.indel.hf.vcf.gz \
-O NA18939.snpindel.hf.vcf.gz
```
