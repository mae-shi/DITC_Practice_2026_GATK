
### 3.　1KG WGSデータ ダウンロード

https://www.internationalgenome.org/data-portal/sample/NA18939

PCR-free high coverageの、SRR1295537をペアでダウンロード (25GBほどある..)

<br>

### 4.　リファレンスのダウンロード

#### DDBJからダウンロード
```
wget https://ddbj.nig.ac.jp/public/public-human-genomes/GRCh38/fasta/Homo_sapiens_assembly38.fasta
```
<br>

#### リファレンスのインデックスの作成

bwa用インデックス(.bwt/.pac/.ann/.amb/.sa )
```
bwa index Homo_sapiens_assembly38.fasta
```

samtools用インデックス (.fai )
```
samtools faidx Homo_sapiens_assembly38.fasta
```

gatk用インデックス (.dict)
```
gatk CreateSequenceDictionary -R Homo_sapiens_assembly38.fasta
```
